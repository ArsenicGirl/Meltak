# Ejecutar proyecto Metalk

Ingresar a la carpeta del proyecto de interfaz
```

```

Descargar dependecias node
```
npm install
```

Hacer correr el servidor (\Meltak\Metalk_interface> npm run dev)
```
npm run dev
```
